✨ Build & Deploy Your Own HTML App with AI + GitHub

Step‑by‑step instructions using Claude, DeepSeek, ChatGPT, and GitHub Pages — no coding experience required.

🧠 Claude

Anthropic's assistant, great for frontend & detailed UI generation.

🌊 DeepSeek

Strong reasoning and coding abilities, free tier available.

💬 ChatGPT

GPT-4 / GPT-3.5 turbo: excellent for generating HTML/CSS/JS quizzes.

🐙 GitHub

Free hosting via GitHub Pages, version control included.

1 📝 Write a clear prompt for the AI
Example: "Create a self-contained HTML/CSS/JS quiz app for STAT103 Statistical Thinking. Include 20 multiple-choice questions per chapter (Ch 0: Introduction, Ch1: Descriptive Stats, Ch2: Probability, Ch3: Inference, Ch4: Ethics). Add a dropdown to switch chapters, show score, check answers, and reset. Style it like a modern learning app."

💡 Tip: Provide syllabus links (RPubs, Quarto book) to let AI extract realistic topics.

2 🧪 Generate & test the HTML code

Copy the AI response (full HTML). Save it as index.html on your computer. Double-click to test locally in your browser. Make tweaks: ask AI to “add a feedback message” or “change color scheme”.

3 🐙 Create a GitHub repository

1. Log into github.com → click New repository.
2. Name it (e.g., stat103-quiz-app), keep it Public (for free hosting).
3. Click Create repository.

4 ⬆️ Upload your index.html file

In the new repo page, click Add file → Upload files. Drag and drop your index.html. Write a short commit message like “Add quiz app”. Click Commit changes.

5 🌐 Enable GitHub Pages

Go to repository Settings → Pages (left sidebar). Under "Branch", select main (or master) and /(root) folder. Click Save. After 1 minute, your app will be live at:
https://<your-username>.github.io/<repository-name>/

6 🔄 Update & iterate with AI

Want more features? Ask AI: “add a timer” or “save progress with localStorage”. Copy new code, upload again to GitHub — changes go live instantly.

⚡ Quick Deployment Cheat Sheet
• Claude/ChatGPT/DeepSeek → generate HTML → save as index.html → GitHub repo → Settings → Pages (branch: main).
• Custom domain: Add CNAME file or configure in Pages settings.
• Privacy: All client-side, no backend required. Perfect for quizzes, portfolios, educational tools.
